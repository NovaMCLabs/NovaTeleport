# Permissions (NovaTeleport)

The authoritative list is `Bukkit/src/main/resources/plugin.yml`.

## Common nodes

- `novateleport.admin` — admin features (includes `/novateleport debug`, stele admin commands, etc.)
- `novateleport.pass` — allows triggering **auto world threshold teleport**
- `novateleport.command.reload` — `/stp reload`

## Economy

- `novateleport.economy.bypass` — bypass all costs

## Animation

- `novateleport.animation.select`
- `novateleport.animation.tech`
- `novateleport.animation.natural`

## Integrations

- Guild: `novateleport.guild.home`, `novateleport.guild.admin`
- Towny: `novateleport.towny.home`, `novateleport.towny.other`
- Toll warps: `novateleport.toll.create`, `novateleport.toll.delete.others`, `novateleport.toll.bypass`
- Teleport log rewind: `novateleport.admin.rewind`
